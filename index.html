About this project

关于我和这个网站的故事

大一刚入学时，我主动参与班委竞选。本想凭借高中三年体育委员的经历竞选文体委员，最后却替补成为了学习委员，而且一做就是四年。

大一上半学年过半，我想着当学习委员也应该做点实事。当时遇到一道名叫“高等数学”的坎，于是开始整理课程资料，后来发现这件事很有意义，也就开启了持续的资料整理。随着课程深入，内容与审美逐渐提升，原来的“复习资料”也更名为更准确的“学习参考资料”。

我一直感谢资料使用者的反馈。毕业之后，如何让更新后的资料继续同步分享与传播，成了新的问题。于是我选择利用 GitHub 搭建这个简单的网站，把课程资料、Python 学习经验和专业工具入口集中整理，作为一个长期维护的开放学习空间。

考试需要一定的技巧，但学习知识不要功利与浮躁。
